Модуль парсера новостей на 1С-Bitrix

Описание
Данный модуль предназначен для парсинга RSS-новостей с сайта https://lenta.ru их в ORM-таблицу. 
Модуль позволяет настраивать параметры парсинга.
Были применены: Bitrix-Framework, ORM, D7, Vue, Bx.Ajax


Установка
Склонировать данный модуль в папку /local/modules/.
Установить модуль на странице "Установленные решения Маркетплейс".

Создать страницу и подключить в ней компонент:
<?$APPLICATION->IncludeComponent(
	"kv:parser.news",
	"",
	Array(
		"SHOW_DATE" => "Y"
	)
);?>

Настройки модуля на странице /bitrix/admin/settings.php?lang=ru&mid=kv.parser


Использование
Переходим в админку на страницу просмотра новостей /bitrix/admin/kv_parser_news_list.php?lang=ru - запускаем парсер.
Проверяем на странице подключения компоненты вывод новостей.
